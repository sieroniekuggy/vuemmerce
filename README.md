# Vuemmerce - Ecommerce Template

## Installing

```
# install dependencies
yarn install

# serve with hot reload at localhost:3000
yarn dev

# prepare for production via webpack
yarn build

# generate static pages via nuxt.js for static hosting
nuxt generate

```
